## About the Project

**Purpose:**
This project was created as an entry for the **GameDev.js Jam 2026**.

**What is this game?**

> A game about the war between two groups, nature lovers and machine scientists...

- **Genre**: A TopDown 2D RPG Game (Zelda-Like)

## Controls (Keybindings)

| Key     | Action     |
|---------|------------|
| `W / ↑` | Move Up    |
| `A / ←` | Move Left  |
| `D / →` | Move Right |

## Tech Stack

- **Language:** JavaScript
- **Framework:** Vanilla JS
- **Game Engine / Game Framework:** Kaplay
- **Runtime:** Node.js
- **Bundler:** Vite
- **Code Editor / IDE:** VSCode + WebStorm
- **Level Design:** Tiled
- **Art:** Aesprite
- **Sounds:** DAW
- **Documenting:** Google Docs + .md files

## Run Locally

```bash
# Clone the repository
git clone "https://github.com/MXD-K1/Naturonics"

# Navigate into the project
cd Naturonics

# Install dependencies
npm install

# Run local development server
npm run dev
```

Open in browser:
```
http://localhost:5173/
```
